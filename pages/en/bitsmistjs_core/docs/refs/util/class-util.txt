====== ClassUtil ======

=====Overview=====

ClassUtil is a utility class that handles class-related tasks such as creating new classes.

=====Methods=====

==== newComponent(superClass, settings, tagName, className) ====

''Static''

Creates a new Function object. Note that the created object **is not an instance**. The component is instantiated by placing a tag specified in the parameter in the HTML file. 

<WRAP admonition note>
This function exists for older browser support. You can use the class notation for modern browsers.
</WRAP>

===Parameters===

^Parameter^Type^Description^
|superClass|Function|The Super class to inherit.|
|settings|Object|A component settings object. This settings will be returned in Component._getSettings().|
|tagName|String|A tag name for this component.|
|className \\ ''optional'' \\ ''Default:(see desc)''|String|A class name for this component. It is guessed from a tag name if this parameter is missing.|
||||

===Return value===

A created function object.

===Examples===

<code javascript>
// Create a search pad component.
var PadSearch = BITSMIST.v1.ClassUtil.newComponent(BITSMIST.v1.Component, {
    "settings": {
        "name": "PadSearch",
        "path": "common"
    }
}, "pad-search");

// Add search() method to the created component.
PadSearch.prototype.search = function(keyword)
{
   ...
};
</code>

<code html>
<html>
...
<body>
<!-- Instantiation -->
<pad-search></pad-search>
</body>
</html>
</code>
==== inherit(subClass, superClass) ====

''Static''

Inherits the subclass from the super class.

===Parameters===

^Parameter^Type^Description^
|subClass|Function|A class that inherits the super class.|
|superClass|Function|A super class.|

===Return value===

undefined

==== createObject(className, ...args) ====

''Static''

Instantiates a class.

===Parameters===

^Parameter^Type^Description^
|className|Function|A class name to instantiate.|
|...args \\ ''optional''| |Arguments to the class constructor.|

===Return value===

A created instance.

==== getClass(className) ====

''Static''

Returns the Function object that has the specified class name.

===Parameters===

^Parameter^Type^Description^
|className|Function|A class name.|

===Return value===

A Function object.
